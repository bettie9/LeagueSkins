## WARNING

Eventough I tried my best to ensure compatibility, some of them may not work properly. I recommend to use the .fantome files with caution, as they may not function as intended or may cause issues within the game. It is advisable to test these assets in practice mode before using them in a live setting.

Also it's important to note that the .fantome files available in this repository are not compatible with the latest versions of cs-lol manager or ltk manager DLL files. It is recommended to use older version of mod-tools.exe and cslol-dll.dll files to ensure compatibility with the .fantome files in this repository. Using incompatible versions may result in errors or issues when trying to use these files.
